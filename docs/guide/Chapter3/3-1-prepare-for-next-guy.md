# 3.1 有关交接前的准备
### 1.有必要写交接流程吗？
**先下结论：**
    *需不需要写交接流程，需要具体情况具体分析。* 

如果您所在的学校有写交接文档的要求，那您必须按照老师的要求写交接流程，那么对于你来说，后面的文章所做的建议就是画蛇点睛。反过来说，您的学校没有要求写交接流程，应与老师先沟通，如果没有获得同意和支持。你写也是画蛇添足。说实话即使你在获得了教师的同意后写出了你的长篇巨著，留在教室的某个角落或讲台的深处也只是吃灰罢了。  

写交接流程，既可复制粘贴，也可倾注心血，应量力而行。  

如果你是我所说的“没有要求写交接流程”的学生，你应该花更多的时间为升学做准备，而不是写一篇不会给升学考试加分且没人看的长篇巨著。(除非你是真的没什么事情可干了。)
### 2.整理好文件
如上所述，如果你的学校要求你做交接，你应该基于学校的要求去做。笔者一下的所有建议都只是在学校需求之外的画龙点睛。如果学校有完整的交接流程，以你的学校要求为准。 

所谓的整理好文件，就是将一些以后新生或者下一个使用此设备的班级用不着的东西删掉。如果设备空间充裕，保险起见您应该只删除试卷及时效性高的文件，及班级特色使用的应用和一切不合适的文件，例如游戏或翻墙软件。 

如果您不知道如何删除应用，请参考如下链接内的文章中的操作方法来删除计算机内安装的应用 

[如何为电脑或大屏幕删除应用](https://cnel.smart-teach.cn/faq/how-to-uninstall-applications-on-pc-or-htpc)

### 3.退出或删除大屏幕所登录的应用程序
#### 1.退出浏览器所登录的全部网站的最快方法
退出浏览器所登录的全部网站的最快方法是清除浏览器的全部cookie，您可以把cookie理解为网站登录的凭证，将凭证清除后浏览器将退出所登录的全部网站。
您可以在搜索引擎中搜索如何清除您所使用的浏览器的全部cookie，下方将以Windows10及以上的默认浏览器edge进行演示清除cookie的方法 

第一步：点击浏览器右上角三个点 

![pic1](/images/docs/public/images/Chapter3/3-1/311.png) 

第二步：点击设置

![pic2](/images/docs/public/images/Chapter3/3-1/312.png)  

第三步：如果有，点击左上角三条杠（若没有请略过这一步）

![pic3](/images/docs/public/images/Chapter3/3-1/313.png)  

第四步：点击”隐私，搜索与服务“按钮

![pic4](/images/docs/public/images/Chapter3/3-1/314.png)  

第五步：下拉，找到cookie按钮并点击

![pic5](/images/docs/public/images/Chapter3/3-1/315.png)  

第六步：点击查看“所有cookie和站点数据”按钮

![pic6](/images/docs/public/images/Chapter3/3-1/316.png)  

第七步：点击“全部删除”按钮即可退出全部网站的登录

![pic7](/images/docs/public/images/Chapter3/3-1/317.png)

####  2.删除即时通讯软件和其他可能泄露隐私的程序并重新安装或彻底删除 

删除qq，微信以及你曾经安装过的程序的时候，切记在卸载时点击删除全部信息（如果有）否则直接删除可能会对您或其他使用过这些程序的人的隐私造成泄露。

![pic8](/images/docs/public/images/Chapter3/3-1/318.png)
