### 电教应有的媒介素养

#### 1.什么是媒介素养

结合电教委员的工作特点和一般媒介素养的概念，`媒介素养`对电教委员而言，可以理解为：电教委员在数字化教育环境中，对各种电子设备、软件工具及信息资源进行有效获取、评估、使用和管理的能力。

具体来说，电教委员的媒介素养应包括：

1. **设备操作能力**：熟练掌握教室多媒体设备（如电子白板、投影仪等）的基本操作和常见故障排查
2. **信息获取与筛选能力**：能够快速从网络获取所需的技术支持信息，并判断其可靠性
3. **工具应用能力**：掌握各种教学辅助软件的安装、配置和使用方法
4. **数字安全意识**：了解基本的网络安全知识，保护设备和信息安全
5. **技术传播能力**：能够将技术知识简洁明了地传达给老师和同学
